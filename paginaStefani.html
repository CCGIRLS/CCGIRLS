Básico sobre o HTML

O que é o html?

A sigla HTML significa Hypertext Markup Language ou Linguagem de Marcação de Hipertexto em português. É usada na contrução de sites para organizar e dar significado para as informações presentes no mesmo.

Algumas tags

Tags Extruturais

  • A tag <header> representa o cabeçalho da página.
  • A tag <footer> representa o rodapé da página
  • A tag <nav> representa que é uma área de navegação da página
  • A tag <aside> define um bloco de conteúdo relacionado ao conteúdo principal. Ex: uma curiosidade, informações sobre o autor, etc.

Algumas tags para formatação

Texto em negrito utilizando a tag <b>. Seu uso não é mais tão indicado pois embora tenha efeito visual esta tag não tm significado semântico.
Texto em itálico utilizando a tag <i>. Mesma questão da tag anterior.
Texto sublinhado utilizando a tag <u>
Texto subscrito utilizando a tag <sub>
Texto sobrescrito utilizando a tag <sup>
Texto com fonte menor do que o padrão utilizando a tag <small>
Texto em itálico utilizando a tag <em>. É a forma indicada para deixar o texto em italico pois esta tag tem o significado semântico de ênfase.
Texto em negrito utilizando a tag <strong>. Mesmo caso da anterior

Tags para listas

Em ambos os casos cada item é representado pela tag <li>

Lista ordenada

Usa a tag <ol>.

  2. Segundo elemento
  3. Terceiro elemento

Lista não ordenada

Usa a tag <ul>.

  • Primeiro elemento
  • Segundo elemento
  • Terceiro elemento

Links

A tag utilizada para links é a <a>, e o endereço para o link é posto no atributo href.

Tabelas

Podemos criar tabelas usando a tag <table>. Cada linha deve ser declarada com a tag <tr> e cada célula com a tag <td>. Opcionalmente podemos usar a tag <th> para declarar células que representem a linha de cabeçalho.

Formulários

O início de um formulário é declarado com a tag <form>. O atributo action recebe o caminho parao arquivo que irá tratar os dados. Já o atributo method recebe o metodo pelo qual os dados serão enviados (POST ou GET).
